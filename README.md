## Working with branches
- Create a new local branch
- Push it to the remote 



Remove file from git

``` git rm myfile.txt ```

## Branch

Create a new branch

``` git checkout -b [my feature branch] ```

Switch to branch

``` git checkout [my branch] ```

Merge branches

- switch to branch A
- merge with branch B  

```
    git checkout branchA 
    git merge branchB
```

Delete branch

``` git branch -d [my branch] ```
